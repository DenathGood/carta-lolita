## ⚠️ IMPORTANTE: Agregar la canción

1. Descarga **death bed - Powfu** en formato MP3
2. Renómbrala exactamente: `musica.mp3`
3. Ponla en la carpeta raíz (junto al index.html)

## 🚀 Cómo subir a GitHub Pages

1. Abre tu repositorio en GitHub
2. Sube TODOS los archivos (index.html, style.css, script.js, musica.mp3 y la carpeta fotos/)
3. Ve a **Settings** → **Pages**
4. En "Source" selecciona **main branch**
5. Guarda — en 1-2 minutos tendrás el link 🎉

El link será algo como:
`https://TU-USUARIO.github.io/NOMBRE-DEL-REPO/`
